# Gerçek Dünya - Landing Page

"The Real World" tarzında bir online eğitim platformu landing page'i.

## Özellikler

- ✅ Modern ve responsive tasarım
- ✅ Next.js 14 App Router
- ✅ TypeScript
- ✅ Tailwind CSS
- ✅ Türkçe içerik
- ✅ Koyu tema (dark mode)
- ✅ 12+ modül showcase
- ✅ Paket fiyatlandırması
- ✅ Smooth scroll animasyonlar

## Kurulum

```bash
# Bağımlılıkları yükle
npm install

# Development server'ı başlat
npm run dev

# Production build
npm run build

# Production server'ı başlat
npm start
```

## Geliştirme

Proje şu anda çalışıyor: http://localhost:3001

## Bileşenler

### Ana Bileşenler
- `Navbar` - Sabit navigation bar
- `HeroSection` - Ana başlık ve CTA
- `WhatIsSection` - Platform tanıtımı
- `ToolsSection` - Özellikler showcası
- `AccessSection` - Erişim avantajları
- `TwoPathsSection` - Karşılaştırma bölümü
- `ServicesSection` - Hizmetler pazarı
- `MotivationSection` - Motivasyon mesajları
- `PricingSection` - Paket seçenekleri
- `KickstartSection` - Başlangıç adımları
- `ModulesSection` - 12+ modül grid'i
- `Footer` - Footer bileşeni

## Teknolojiler

- **Framework:** Next.js 14.2.0
- **Language:** TypeScript 5.0
- **Styling:** Tailwind CSS 3.4.0
- **React:** 18.3.0
